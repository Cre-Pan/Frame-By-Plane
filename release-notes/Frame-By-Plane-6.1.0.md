# 🎞️ Frame By Plane — 6.1.0 LTS

**Frame By Plane 6.1 LTS focuses on a stable native animation workflow, a broader effects system, cleaner project controls and lighter downloads for every supported platform.**

## ✨ Main highlights

- 🎬 More reliable native image and image-sequence playback.
- 🗂️ Refined single-plane, folder, multiplane, PSD and Procreate imports.
- 🎨 **62 built-in effects** across distortion, color, blur, stylization, masks and utilities.
- 🌓 Improved layer blend modes, alpha-aware masks and effect ordering.
- ✂️ Stronger cutout, outline and thickness workflows.
- 🎥 Refined camera, project, render and sequence controls.
- 🧪 Autonomous developer tests and stricter release-gate validation.
- 📦 Smaller platform-specific packages with only the required Python wheels.
