# Frame by Plane v2.3.13

Reliability update focused on Color Plane orientation, native Blender duplicate/delete behavior, image-sequence import, and Holdout toggles.

## Highlights

- Color Plane now supports **Vertical / Horizontal** orientation.
- Native `Shift + D` duplication now repairs linked rig/image-plane setups.
- Native `X` delete now cleans the associated image plane when deleting an FBP rig.
- Single Plane import now treats multiple selected images as one animated sequence.
- Folder/layer Holdout toggles are now reversible and safer.

## Notes

This GitHub package contains the source folder version of the addon.
For Blender Extensions upload, use the Blender extension package build format instead.
