# 🎞️ Frame By Plane

**Frame By Plane works like Blender’s native “Image as Plane”… but built for animation, image sequences, layered scenes and 2.5D workflows.**

Frame By Plane is a Blender add-on for **2D animation, 2.5D animation, image planes, image sequences, video planes, multiplane camera setups, parallax, cut-out animation, motion graphics, compositing, background art and non-destructive image styling**.

Import images, sequences, videos and folder-based projects as controllable rigged planes, then animate, organize, mask, distort and stylize every layer independently.

Although Frame By Plane was designed primarily for animation, it is also useful for:

- still-image editing and stylization inside Blender;
- layered background design;
- anime and illustrated environments;
- 2.5D parallax scenes;
- motion graphics;
- projections and scenic elements;
- image-based installations;
- cut-out artwork;
- compositing and masking;
- static multiplane illustrations;
- camera-based depth effects.

From **Lotte Reiniger’s silhouette worlds**, to the **Disney multiplane camera**, the dimensional setups of **Fleischer Studios**, anime backgrounds, European cut-out animation and experimental cinema, Frame By Plane brings the same layered philosophy into Blender:

**separate layers, independent motion, editable timing, camera depth, parallax and creative freedom.**

[Install from Blender Extensions](https://extensions.blender.org/add-ons/frame-by-plane/)

## 🏛️ Inspired by animation history

Frame By Plane is inspired by the layered construction of traditional animation.

Before digital compositing, artists created depth by separating backgrounds, characters, foreground elements, shadows and effects onto different physical layers. Each layer could be moved independently, placed at a different distance from the camera or replaced frame by frame.

This approach appears throughout animation history:

- **Lotte Reiniger** and silhouette animation;
- **Walt Disney’s multiplane camera**;
- **Fleischer Studios** and dimensional tabletop setups;
- American cel animation;
- Japanese anime background layering;
- French, Italian and European cut-out traditions;
- experimental stop-motion;
- collage animation;
- mixed-media cinema.

Frame By Plane translates that layered logic into a modern Blender workflow for **depth, parallax, camera movement, image sequences and animated 2D/2.5D scenes**.

---

## ✨ What makes it different?

Blender can already import an image as a plane. Frame By Plane turns that simple idea into a complete system for managing animated images, layered backgrounds and multiplane projects.

With Frame By Plane you can:

- import single images, image sequences and video files;
- scan folders and nested project structures;
- generate rigged planes with dedicated selectable controllers;
- animate every layer independently;
- edit individual frames directly from a Frame List;
- reorder, duplicate, reverse, split and merge frame sequences;
- use custom FPS and timing controls;
- choose Loop, Ping-Pong or One Shot playback;
- hold the first and last frame outside the active animation range;
- fit planes automatically to the active camera;
- create Color, Gradient, Cutout and Holdout planes;
- crop and extend images non-destructively;
- organize planes into collections and collapsible layer groups;
- control visibility, opacity, selection, locking and holdout states;
- apply 2D shader effects and 3D Geometry Nodes effects;
- add masks to complete layers or individual effects;
- build complete multiplane scenes from folders;
- create parallax and 2.5D camera movement;
- save compatible node groups as reusable custom effects;
- access common tools through a custom `Z` Pie Menu.

## 🎨 Planes for every workflow

### 🖼️ Image Plane

Import a single still image as a rigged Blender plane. Useful for backgrounds, props, illustrations, textures, matte paintings and static 2.5D compositions.

### 🎞️ Image Sequence Plane

Import a folder of numbered frames as a native Blender image sequence. Control playback, FPS, start frame, frame holds, looping and individual frame order directly from the Frame By Plane interface.

### 🎥 Video Plane

Import a video file as an animated rigged plane using Blender’s native movie-image system. Useful for projections, reference footage, compositing elements and mixed-media scenes.

### ✂️ Cutout Plane

Create an alpha-aware cut-out plane from images, sequences or videos. Designed for silhouette animation, character parts, paper-cut animation and image-based puppets.

### 📁 Multiplane Project

Import multiple folders, images, sequences and videos as separate layers or organized collections. Frame By Plane detects common frame-numbering patterns, preserves folder hierarchy and prepares the project inside the Multiplane Setup before generation.

### 🎨 Color Plane

Create solid-color planes and animated color frames directly inside Blender. Useful for graphic shapes, mattes, backgrounds and compositing elements.

### 🌈 Gradient Plane

Create linear or radial gradients with editable colors, transparency and animation controls.

### ⚫ Holdout Plane

Create procedural holdout layers for masking, silhouettes and compositing-style workflows.

## ⏱️ Image-sequence timing and frame controls

Frame By Plane adds a dedicated Frame List for editing animated image planes.

You can:

- change the order of individual frames;
- select and move multiple frames together;
- reverse selected frames;
- duplicate frames;
- insert transparent frames;
- split or merge sequences;
- set custom timing and FPS;
- offset playback;
- define the animation starting point;
- use Loop, Ping-Pong and One Shot modes;
- hold the first frame before playback;
- hold the final frame after playback;
- preview thumbnails;
- jump directly to a frame from the list.

Playback uses Blender’s native image, image-sequence and movie datablocks while Frame By Plane manages timing, materials, frame order and layer controls.

## 🧩 Custom effects

Frame By Plane can turn compatible local Shader Node and Geometry Nodes groups into reusable custom effects with custom names, descriptions, categories, icons and exposed controls.

The original node group remains the source of truth, allowing one node setup to be reused across multiple layers.

## 🎥 Multiplane cameras and parallax

Frame By Plane is designed for layered camera compositions.

You can:

- distribute layers across depth;
- create foreground, middle-ground and background groups;
- generate a camera directly from the setup;
- use perspective or orthographic projection;
- fit selected planes to the camera;
- create parallax through camera or layer movement;
- preserve the apparent scale of camera-facing elements;
- frame imported planes automatically in the viewport;
- use a fully black camera Passepartout outside the frame.

The result is a workflow suitable for traditional multiplane animation, anime backgrounds, illustrated environments and 2.5D scene construction.

## ⚙️ Native Blender image and video playback

Frame By Plane uses Blender’s native image, image-sequence and movie systems.

Animated planes are built around `ShaderNodeTexImage`, native image users and Blender materials, while Frame By Plane adds frame-list management, playback modes, timing controls, non-destructive effects, masks, rig controls, collection organization and project-import tools.

This keeps scenes compatible with Blender’s standard rendering, material and animation systems.

## 🚀 Performance and project safety

Frame By Plane includes:

- cached layer and effect lookups;
- optimized viewport updates;
- lightweight native playback;
- separate viewport, playback and render quality for demanding effects;
- prevention of duplicate modifiers and node groups;
- safe handling and relinking of missing media;
- project-health checks;
- effect profiling;
- lifecycle and backend diagnostics;
- cleanup of safe orphaned Frame By Plane data.

Heavy Geometry Nodes effects such as Felt Fuzz, Text Matrix, high-resolution Extrude or complex deformation can still require additional viewport and render resources.

## 📦 Installation

### Blender Extensions

Install the latest public release from the official Blender Extensions page:

[Frame By Plane on Blender Extensions](https://extensions.blender.org/add-ons/frame-by-plane/)

### Install from a GitHub release

1. Download `frame_by_plane-5.8.6.zip` from the GitHub release assets or from the `release/` folder in this repository.
2. Open Blender.
3. Go to **Edit → Preferences → Get Extensions**.
4. Open the menu in the top-right corner.
5. Choose **Install from Disk**.
6. Select the ZIP without extracting it.

> The GitHub repository source ZIP is not the installable extension package. Use the ZIP stored in `release/`.
